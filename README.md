# 💰 Тихий Финансовый Контролёр v2

Telegram-бот для учёта финансов в группе. Работает тихо — не пишет в чат, отчитывается только в личку.

---

```

├── bot.py           # Главный файл
├── database.py      # SQLite: транзакции, настройки
├── requirements.txt
├── railway.toml
├── .env.example
└── .gitignore
```

---

## 🚀 Деплой на Railway

### 1. Создайте бота
- [@BotFather](https://t.me/BotFather) → `/newbot` → скопируйте токен
- Выключите Privacy Mode: `Bot Settings → Group Privacy → Turn off`

### 2. Узнайте ID
| Что | Как |
|-----|-----|
| Ваш `ADMIN_ID` | [@userinfobot](https://t.me/userinfobot) |
| `ALLOWED_GROUP_ID` | Добавьте [@getidsbot](https://t.me/getidsbot) в группу |

### 3. Загрузите на GitHub и подключите Railway
```bash
git init && git add . && git commit -m "init"
git remote add origin https://github.com/ВАШ_НИК/finance-bot.git
git push -u origin main
```
Railway → New Project → Deploy from GitHub repo

### 4. Переменные окружения в Railway
```
BOT_TOKEN          = токен
ADMIN_ID           = ваш ID
ALLOWED_GROUP_ID   = ID группы (с минусом)
ALLOWED_USER_IDS   = другие пользователи через запятую
DB_PATH            = /data/finance.db
```

### 5. Постоянный том (Volume)
Railway → Settings → Volumes → Add Volume → Mount Path: `/data`

## 📖 Синтаксис записей в группе

| Сообщение | Результат |
|-----------|----------|
| `+600$ 12200` | +600 USD, коммент «12200» |
| `+28.297.000 QAHRAMON` | +28 297 000 UZS |
| `-150000 аренда` | -150 000 UZS |
| `50000` | -50 000 UZS (без знака = расход) |
| `-50$` | -50 USD |

## 💬 Команды в личных сообщениях

### Для всех разрешённых пользователей
| Команда | Описание |
|---------|---------|
| `/start` `/balance` | Текущий баланс + последние 5 операций |
| `/report` | Отчёт за текущий месяц |
| `/report сегодня` | За сегодня |
| `/report неделя` | За текущую неделю |
| `/report месяц` | За текущий месяц |
| `/report 01.06-30.06` | За произвольный период |
| `/help` | Список команд |

### Только для администратора
| Команда | Описание |
|---------|---------|
| `/delete 42` | Удалить запись #42 |
| `/edit 42 текст` | Изменить комментарий записи #42 |
| `/setstart 01.07.2025` | Учитывать записи только с этой даты |
| `/setstart сброс` | Убрать ограничение по дате |

## 📅 Функция «Начать с определённой даты»

Если группа уже работала и вы хотите считать баланс только с сегодняшнего дня:

1. Напишите боту в личку: `/setstart 30.03.2026`
2. Бот подтвердит: «Дата начала учёта установлена: 2026-03-30»
3. Старые сообщения из группы будут игнорироваться
4. Баланс и отчёты будут считаться только с этой даты

Чтобы вернуть учёт всех записей: `/setstart сброс`

## 🗑 Удаление и редактирование

При каждой новой записи бот присылает уведомление с кнопкой **«Удалить»** — можно удалить сразу.

Позже можно удалить по ID: `/delete 42`

ID виден в уведомлениях и в отчётах.
